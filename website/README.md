# Empire Connect Website

Static marketing site for Empire Connect.

## Files

- `index.html` contains the main landing page
- `support.html` contains the support hub
- `privacy.html` contains the privacy policy
- `terms.html` contains the terms of use
- `404.html` contains the fallback page
- `styles.css` contains the shared visual system
- `script.js` contains mobile nav and reveal effects
- `assets/` contains the bundled banner and promo video

## Deploy

Deploy the contents of this folder to any static host such as GitHub Pages, Netlify, Vercel static hosting, Cloudflare Pages, or an S3-backed site.

## Notes

- The current featured promo video is `assets/empire-360.mp4`.
- The site uses Google Fonts, so the host needs normal public internet access for those font requests.
